# Pasta de Assets de Produção — Mãos na Palavra™

Esta é a pasta oficial onde o código do site (`index.html` e `style.css`) busca as imagens. Quando você tiver as imagens finais prontas, salve-as nesta pasta respeitando os nomes de arquivo descritos abaixo para que elas apareçam automaticamente no site.

## 📂 Estrutura de Subpastas e Nomes de Arquivos:

* `assets/og_image_share.jpg` — Imagem que aparece no compartilhamento de link (WhatsApp/Facebook).
* `assets/hero_composition.webp` — Imagem principal da Hero (Desktop).
* `assets/child_coloring_hero.webp` — Imagem da criança pintando na Hero (Mobile).
