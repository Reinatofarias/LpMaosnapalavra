# Referências de Design e Inspiração

Utilize este diretório para colocar as imagens de referência, inspirações visuais, ilustrações de mascotes ou capturas de tela de páginas que você gosta.

## 📂 Como usar:
1. Cole as imagens de referência neste diretório (`e:\Downloads\Mãos na Palavra LP\references\`).
2. Você pode usar formatos comuns como `.png`, `.jpg`, `.jpeg` ou `.webp`.
3. Assim que colocar as imagens aqui, me avise! Eu poderei analisar as referências e reescrever o arquivo `style.css` e o `index.html` para aplicar um design muito mais maduro, premium, moderno e alinhado com a sua visão.
